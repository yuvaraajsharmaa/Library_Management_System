# Library Management System – Group 6

## Features
- Book Management (Add, Update, Remove, Search by Title/Author/ISBN)
- Member Management (Register Standard/Premium, Update Info, View History)
- Borrow/Return Books with due date (14 days)
- **Premium Membership**: borrow up to 6 books (Standard = 3)
- **Availability Tracking**: shows when a borrowed book will be available
- Input validation (ISBN, email, positive IDs)
- Error handling and console menus
- JUnit 5 tests

## Technologies
- Java 11+
- JUnit 5
- VS Code / IntelliJ

## Setup & Run
1. Clone the repository.
2. Open in VS Code (ensure Java Extension Pack installed).
3. Compile and run `src/ui/Main.java`.
